Lazyman config: StartBase

StartBase Neovim Configuration Information

Small configuration that includes a plugin manager

  • Install and initialize: lazyman -x StartBase
  • Configuration category: Starter
  • Base configuration: Custom
  • Plugin manager: Lazy
  • Installation location: ~/.config/nvim-StartBase

Links to all Lazyman supported configuration documents

StartBase Keymaps

normal mode keymaps

Description LHS RHS
bl buffer #
bq bdelete
w write
a :keepjumps normal! ggVG
l g_
h ^
% (MatchitNormalForward)
Nvim builtin & :&&
Nvim builtin Y y$
[% (MatchitNormalMultiBackward)
]% (MatchitNormalMultiForward)
gx NetrwBrowseX
g% (MatchitNormalBackward)
gp “+p
gy “+y
x “_x
NetrwBrowseX :call netrw#BrowseX(netrw#GX(),netrw#CheckIfRemote(netrw#GX()))
(MatchitNormalMultiForward) :call matchit#MultiMatch(“W”, “n”)
(MatchitNormalMultiBackward) :call matchit#MultiMatch(“bW”, “n”)
(MatchitNormalBackward) :call matchit#Match_wrapper(’‘,0,’n’)
(MatchitNormalForward) :call matchit#Match_wrapper(’‘,1,’n’)
Nvim builtin nohlsearch
